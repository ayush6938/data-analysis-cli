Data Analysis CLI Tool

A powerful command-line based data analysis system built using Python.
This tool analyzes datasets, assigns grades, detects risk levels, and generates insights automatically.

---

🚀 Features

- 📈 Analyze datasets efficiently
- 🏆 Assign grades (A / B / C)
- ⚠️ Risk detection (Low / Stable / High)
- 💡 Generate smart insights
- 📄 Save reports automatically
- 🔄 Config-driven system using JSON

---

🛠 Tech Stack

- Python
- JSON (Configuration handling)
- CLI (Command Line Interface)
- Modular Programming

---

▶️ How to Run Project

Just run python main.py
When you run main.py, you will get start menu.
You can select your desired options from options.
